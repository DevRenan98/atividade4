Atividade 1

Seguindo os passos da aula, livros indicados e, a leituras recomendadas;

Desenvolva um formulário WEB de sua preferência que contenha os campos exibidos e monte um texto com as seleções do usuário.

O formulário deve ter no mínimo:

3 input tipo radio ;

5 input tipo checkbox;

2 input texto;

2 input numéricas;

c. Além disso o formulário deve mudar as cores com base em combinações de seleção diferentes: Exemplo: Background da página fica ROSA se o usuário digitar o nome "Maria", digitar um valor maior que 10 em um campo numérico e selecionar todas as checkbox. (Você também pode mudar cor de: text, cor de uma seção ou div, imagem de fundo... Seja criativo!)

d. Aplique CSS com Bootstrap para posicionamento e tipografia de elementos conforme a aula anterior.
